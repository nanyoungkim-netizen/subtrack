📋 자피어 설정 방법 (5단계)

1
zapier.com 접속 → Create Zap 클릭
2
Trigger: Slack 선택
Event: New Message Posted to Channel
Channel: #구매-요청 선택
3
Filter 추가 (자동-결제 워크플로만 필터링)
조건: 메시지 텍스트에 자동-결제 또는 워크플로 봇 이름 포함
4
Action: Webhooks by Zapier 선택
Event: POST
URL: 위의 웹훅 URL 붙여넣기
Payload Type: JSON
5
Data 필드 매핑 (슬랙 워크플로 필드 → SubTrack 필드)
SubTrack 필드명슬랙 워크플로 값예시
service서비스명 워크플로 필드Figma
user사용자/담당자 필드Rami
amount금액 필드20달러
payment결제수단 필드1234
cycle결제주기 필드월결제
start_date시작일 필드2026-03-10
description설명/내용 필드디자인 툴
note비고 필드3개월 사용 예정
